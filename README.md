####百味免费图库插件
---
#####该插件让你直接在编辑器页面上传图片到免费服务器
#####上传完成后自动在编辑器中添加图片，不需要设置全站CDN
#####功能简单，更方便

###安装
---
####1. 上传 `free-tuchaung`目录 到 `/wp-content/plugins/` 目录
####2. 在后台插件菜单激活该插件
####3. 在后台设置设置

###问题
---

#####不能上传？

>上传前要设置ak和sk还有bucket
不要有多余的 “/”哦

#####自动添加的图片title,alt信息？

>信息默认为文章标题，如果标题为空则为空

####为什么我上传的图片不显示？

>请确定编辑器是在可视化下，文本模式不能正常添加链接
