## 百味免费图库插件
---
* 该插件让你直接在编辑器页面上传图片到免费服务器
* 上传完成后自动在编辑器中添加图片
* 功能简单方便

安装
---
* 1. 上传 `free-picture-links`目录 到 `/wp-content/plugins/` 目录
* 2. 在后台插件菜单激活该插件
* 3. 在后台设置设置
